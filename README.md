## 📖 About

**NooB Player** is a cross-platform desktop video player built using [Electron.js](https://www.electronjs.org/) and plain Vanilla JavaScript. It provides a clean, distraction-free interface for playing your local video files — no bloat, no subscriptions, just play.

> 🎯 Built as a personal project to explore Electron's capabilities in building native-like desktop apps with web technologies.

---

## ✨ Features

- ▶️ **Play / Pause & Seek** — Smooth playback controls with an intuitive seekbar
- 📂 **Playlist Support** — Queue up multiple video files and navigate between them
- 💬 **Subtitle Support** — Load and display external subtitle files (.srt, .vtt)
- ⌨️ **Keyboard Shortcuts** — Control playback without ever touching your mouse
- 🖥️ **Cross-Platform** — Works on Windows, macOS, and Linux
- 🌙 **Minimal UI** — Clean, dark-themed interface that keeps the focus on your content

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone [https://github.com/your-username/noob-player.git](https://github.com/NoobZ9784/NooB_Player.git)
   cd noob-player
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run in development mode**
   ```bash
   npm start
   ```

### Build for Production

To package the app for your platform:

```bash
# For Windows
npm run build:win

# For macOS
npm run build:mac

# For Linux
npm run build:linux
```

> Packaged installers will be available in the `/dist` folder.

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `→` | Seek forward 5 seconds |
| `←` | Seek backward 5 seconds |
| `↑` | Volume up |
| `↓` | Volume down |
| `M` | Mute |
| `F` | Toggle fullscreen |
| `N` | Next track (playlist) |
| `Shift + N` | Previous track (playlist) |
| `R` | Rotate Right |
| `Shift + R` | Rotate Left |
